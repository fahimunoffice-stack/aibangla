## [1.0.0] - 2025-04-02

### Added
- Initial release of AiBangla
- AI-powered Bengali phonetic correction using Groq API
- Real-time translation to any language
- System tray integration with toggle functionality
- Cross-platform support (Windows, Linux, macOS)
- Multiple AI models (Llama 3.1, Mixtral, Gemma 2)
- Configurable settings with JSON persistence
- Response caching to avoid duplicate API calls
- Fallback phonetic mapping when API fails
- GitHub Actions CI/CD for automated builds and releases
